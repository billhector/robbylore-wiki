# Robby Lore — Public Wiki

Source repo for [robbylore.org](https://robbylore.org), a fan-built encyclopedia of the comedian Robby Hoffman.

Built with [Quartz](https://quartz.jzhao.xyz/), deployed to Cloudflare Pages on every push.

## Contributing

Corrections + additions welcome: open a PR or email robbyhoffmanlore@gmail.com.

Authoring happens in [Obsidian](https://obsidian.md) — clone this repo, point Obsidian at `content/`, edit, push.

## Local preview

```
npm install
npx quartz plugin install
npx quartz build --serve
```

Opens at localhost:8080.
